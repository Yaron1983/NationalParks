# Chat Frontend - React Application

Frontend application for chat rooms using React and WebSockets.

## Installation

```bash
npm install
```

## Development

```bash
npm start
```

The app will run on http://localhost:3000

## Build

```bash
npm run build
```

## Configuration

Set the API base URL using environment variable:

```bash
REACT_APP_API_URL=http://localhost:8000/api/chat npm start
```

## Features

- View all available chat rooms
- Create new chat rooms
- Real-time messaging using WebSockets
- Connection status indicator
- Responsive design
